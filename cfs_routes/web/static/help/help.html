How to Use
  1. Select a departure airport from the dropdown and click Add — repeat for each departure airport you need.
  2. In a row's destination field, type an ICAO code (e.g. CYYC) or a cardinal direction (e.g. NE), then press Enter or click the search button.
  3. The modal shows all mandatory IFR routes for that pair, grouped by altitude (HIGH / LOW). When no specific route exists for a destination, all cardinal direction departure routes from that airport are shown instead.
Keyboard Shortcuts
Ctrl + K Open the airport search dropdown
↑ / ↓ Move between destination fields
Leader Key

Press Esc to activate the leader key, then within 1.5 s press a digit to focus that row's destination field.

Esc → 1–0 Focus destination field for row 1–10
